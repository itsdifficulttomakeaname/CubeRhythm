# 面板
## 游戏内面板
每一面都显示一样的内容，但是注意朝向\
以正面为例(显示为TextDisplay，注意位置，朝向为垂直判定面方向)
```
连击数(小字体)               分数(小字体)










歌曲名(小字体)               难度(小字体)
```
## 结算时面板
只在正面展示
```



     评级(大字体，根据具体等级修改颜色)     准确度(中字体，黄色)
     分数(中字体，黄色)                  难度(中字体，根据具体难度显色)
                    EXACT数量(小字体，蓝色)
                     JUST数量(小字体，黄色)
                 "("EARLY数量(小字体，黄色)")"
                 "("LATE数量(小字体，黄色)")"
                     MISS数量(小字体，红色)




```
其中"("和")"表示要显示这个字符

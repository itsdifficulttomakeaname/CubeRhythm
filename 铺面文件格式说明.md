## 当前格式说明

### 文件结构
当前铺面使用两个 `.sk` 文件配对：

1. **属性文件** (`*_properties.sk`)
   - 定义谱面元数据
   - 注册谱面到系统

2. **内容文件** (`*.sk` 或 `-*.sk`)
   - 定义音符数据
   - `-` 前缀表示禁用状态

### 当前格式示例

#### 属性文件 (simpletone_properties.sk)
```skript
options:
    谱面: simpletone
    曲名: simpletone
    曲师: CRE
    谱师: PiraTom
    难度: &bTutorial 1
    时长: 51
    偏移: 0
    BPM: 130

on load:
    registerChart("{@谱面}", "{@曲名}", "{@难度}", "{@曲师}", "{@谱师}", {@时长}, {@偏移})
    setBPM("{@谱面}", {@BPM})
```

#### 内容文件 (simpletone.sk)
```skript
on load:
    delete {loadedNotes::*}
    set {maxNote} to 0
    set {sound} to "cr.simpletone"
    loadChart()

function loadChart():
    # 教程剧情脚本
    execution(0, {title1})

    # 音符定义
    tap(14.77, "w", 1, 0, false, "")
    hold(29.55, "w", 2, 1, false, "")
    drag(42.46, "w", 2, 1, false, "")
    flick(36.92, "w", "left", false, "")
    double(22.15, "w", -1, -1, -2, 1, false, "")
```

## 音符类型详解

### 1. Tap 音符 (点击音符)
**功能**: 与判定面重合时单击鼠标

**Skript格式**:
```skript
tap(time: number, face: text, x: number, y: number, glowing: boolean, tag: text)
```

**JSON格式**:
```json
{
  "type": "tap",
  "time": 14.77,
  "face": "w",
  "position": { "x": 1, "y": 0 },
  "glowing": false,
  "tag": ""
}
```

**参数说明**:
- `time`: 音符触发时间（秒）
- `face`: 判定面方向（w=前，a=左，s=后，d=右）
- `position.x`: X坐标（-3 到 3）
- `position.y`: Y坐标（-3 到 3）
- `glowing`: 是否发光
- `tag`: 自定义标签

---

### 2. Hold 音符 (长按音符)
**功能**: 与判定面重合时按住鼠标右键

**Skript格式**:
```skript
hold(time: number, face: text, x: number, y: number, glowing: boolean, tag: text)
```

**JSON格式**:
```json
{
  "type": "hold",
  "time": 29.55,
  "face": "w",
  "position": { "x": 2, "y": 1 },
  "glowing": false,
  "tag": ""
}
```

**特点**:
- 连续的hold音符形成长按轨道
- 按住后无需对准
- 渲染为白色方块

---

### 3. Drag 音符 (拖动音符)
**功能**: 与判定面重合时对准即可，无需按键

**Skript格式**:
```skript
drag(time: number, face: text, x: number, y: number, glowing: boolean, tag: text)
```

**JSON格式**:
```json
{
  "type": "drag",
  "time": 42.46,
  "face": "w",
  "position": { "x": 2, "y": 1 },
  "glowing": false,
  "tag": ""
}
```

**特点**:
- 连续的drag音符形成拖动轨迹
- 无需按键，准星对准即可
- 渲染为黄色方块

---

### 4. Flick 音符 (滑动音符)
**功能**: 按箭头方向旋转90°到邻近判定面

**Skript格式**:
```skript
flick(time: number, face: text, turn: text, glowing: boolean, tag: text)
```

**JSON格式**:
```json
{
  "type": "flick",
  "time": 36.92,
  "face": "w",
  "turn": "left",
  "glowing": false,
  "tag": ""
}
```

**参数说明**:
- `turn`: 滑动方向（"left" 或 "right"）

**特点**:
- 位置固定在判定面中心
- 渲染为白色玻璃板 + 箭头
- 需要玩家快速旋转视角

---

### 5. Double 音符 (双音符)
**功能**: 同时击打两个位置，双击鼠标左右键

**Skript格式**:
```skript
double(time: number, face: text, x1: number, y1: number, x2: number, y2: number, glowing: boolean, tag: text)
```

**JSON格式**:
```json
{
  "type": "double",
  "time": 22.15,
  "face": "w",
  "positions": [
    { "x": -1, "y": -1 },
    { "x": -2, "y": 1 }
  ],
  "glowing": false,
  "tag": ""
}
```

**参数说明**:
- `positions`: 包含两个位置的数组

**特点**:
- 渲染两个橙色方块
- 两个音符之间有白色连线
- 需要同时击中才算成功

---

### 6. Execution (剧情触发器)
**功能**: 在指定时间执行脚本段落

**Skript格式**:
```skript
execution(time: number, section: section)
```

**JSON格式**:
```json
{
  "type": "execution",
  "time": 0,
  "section": "title1",
  "description": "显示欢迎标题"
}
```

**特点**:
- 非音符类型
- 用于触发剧情、教程提示等
- 不计入总音符数

## 音符读取流程

### 1. 数据加载流程

```mermaid
graph TD
    A[加载铺面] --> B[读取metadata]
    B --> C[初始化{loadedNotes}]
    C --> D[遍历音符定义]
    D --> E[调用音符函数]
    E --> F[异步注册音符]
    F --> G[存储到{loadedNotes::*}]
    G --> H[增加音符ID]
    H --> D
    D --> I[加载完成]
```

### 2. 音符注册机制

**位置**: `load.sk` 第7-62行

```skript
create section stored in {asyncNoteRegister}:
    if {editMode} is not true:
        # 游戏模式：存储到{loadedNotes}
        set {loadedNotes::type::%{_id}%} to {_type}
        set {loadedNotes::time::%{_id}%} to {_time}
        set {loadedNotes::face::%{_id}%} to {_face}
        # ... 其他属性
    else:
        # 编辑器模式：存储到{editorSave}
        set {_uuid} to random uuid
        set {editorSave::type::%{_uuid}%} to {_type}
        # ... 其他属性
```

**对应JSON处理逻辑**:
```json
{
  "loadingProcess": {
    "step1": "解析JSON文件",
    "step2": "验证格式版本",
    "step3": "加载metadata到内存",
    "step4": "遍历notes数组",
    "step5": "按类型分发音符数据",
    "step6": "按时间排序",
    "step7": "准备渲染队列"
  }
}
```

### 3. 内存数据结构

**当前Skript变量结构**:
```
{loadedNotes::type::1} = "tap"
{loadedNotes::time::1} = 14.77
{loadedNotes::face::1} = "w"
{loadedNotes::x::1} = 1
{loadedNotes::y::1} = 0
{loadedNotes::glowing::1} = false
{loadedNotes::tag::1} = ""
```

**对应JSON内存模型**:
```json
{
  "loadedNotes": {
    "1": {
      "type": "tap",
      "time": 14.77,
      "face": "w",
      "x": 1,
      "y": 0,
      "glowing": false,
      "tag": ""
    }
  }
}
```

### 4. 音符排序与播放

**位置**: `main.sk` 第225-256行

**当前流程**:
1. 按时间排序所有音符
2. 根据当前播放时间判断是否生成音符
3. 调用对应的draw函数渲染音符

**JSON格式下的优势**:
```json
{
  "optimizations": {
    "preSort": "JSON可在加载时预排序",
    "indexing": "支持时间轴索引",
    "batching": "可批量加载同一时间的音符",
    "streaming": "支持流式加载大型铺面"
  }
}
```

## JSON Schema 定义

### 完整Schema

```json
{
  "$schema": "http://json-schema.org/draft-07/schema#",
  "title": "Main Chart Format",
  "type": "object",
  "required": ["version", "metadata", "notes"],
  "properties": {
    "version": {
      "type": "string",
      "pattern": "^\\d+\\.\\d+\\.\\d+$"
    },
    "metadata": {
      "type": "object",
      "required": ["id", "title", "artist", "charter", "bpm"],
      "properties": {
        "id": { "type": "string" },
        "title": { "type": "string" },
        "artist": { "type": "string" },
        "charter": { "type": "string" },
        "difficulty": {
          "type": "object",
          "properties": {
            "name": { "type": "string" },
            "color": { "type": "string" },
            "level": { "type": "number" }
          }
        },
        "audio": { "type": "string" },
        "duration": { "type": "number" },
        "offset": { "type": "number" },
        "bpm": { "type": "number" },
        "tags": {
          "type": "array",
          "items": { "type": "string" }
        }
      }
    },
    "events": {
      "type": "array",
      "items": {
        "type": "object",
        "required": ["type", "time"],
        "properties": {

            "enum": ["w", "a", "s", "d"]

            "type": "object",

            "type": "array",
```

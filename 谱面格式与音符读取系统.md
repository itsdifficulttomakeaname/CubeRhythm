## 当前谱面格式 (Skript)

### 文件结构

每个谱面由两个文件组成：

#### 1. 谱面属性文件 `charts\谱面名_properties.sk`

```skript
on load:
    registerChart("simpletone", "Simple Tone", "§aLv.1", "Artist Name", "Charter Name", 60, 0)
    setBPM("simpletone", 120)
```

**参数说明：**
- `id`: 谱面唯一标识符
- `name`: 歌曲名称
- `level`: 难度等级（支持颜色代码）
- `artist`: 作曲家
- `charter`: 谱师
- `length`: 歌曲时长（秒）
- `offset`: 谱面偏移（毫秒）
- `bpm`: 每分钟节拍数

#### 2. 谱面数据文件 `charts\谱面名.sk`

```skript
on load:
    # Tap 音符
    tap(1.5, "w", 0, 0, false, "")
    tap(2.0, "a", 1, -1, false, "intro")

    # Hold 音符
    hold(3.0, "s", 0, 1, false, "")

    # Drag 音符
    drag(3.5, "d", -1, 0, false, "")

    # Flick 音符
    flick(4.0, "w", "left", false, "")
    flick(4.5, "w", "right", true, "highlight")

    # Double 音符
    double(5.0, "a", -1, 0, 1, 0, false, "")

    # Execution 音符（执行自定义脚本）
    create section stored in {customEffect1}:
        # 自定义代码
        send "Custom effect triggered!" to all players
    execution(6.0, {customEffect1})
```

### 音符函数参数详解

#### Tap（点击音符）
```skript
tap(time, face, x, y, glowing, tag)
```
- `time`: 音符时间（秒）
- `face`: 判定面 ("w"前/"a"左/"s"后/"d"右)
- `x`: X坐标 (-3 到 3)
- `y`: Y坐标 (-3 到 3)
- `glowing`: 是否发光 (true/false)
- `tag`: 标签（用于特殊效果标记）

#### Hold（长按音符）
```skript
hold(time, face, x, y, glowing, tag)
```
参数同 Tap

#### Drag（滑键音符）
```skript
drag(time, face, x, y, glowing, tag)
```
参数同 Tap

#### Flick（滑动音符）
```skript
flick(time, face, turn, glowing, tag)
```
- `time`: 音符时间（秒）
- `face`: 判定面
- `turn`: 转向方向 ("left"/"right")
- `glowing`: 是否发光
- `tag`: 标签

#### Double（双键音符）
```skript
double(time, face, x1, y1, x2, y2, glowing, tag)
```
- `time`: 音符时间（秒）
- `face`: 判定面
- `x1, y1`: 第一个音符坐标
- `x2, y2`: 第二个音符坐标
- `glowing`: 是否发光
- `tag`: 标签

#### Execution（执行音符）
```skript
execution(time, section)
```
- `time`: 执行时间（秒）
- `section`: Skript section 对象

## 音符读取流程

### 当前实现 (Skript)

```
伪代码：

// 1. 谱面列表刷新 (gui.sk:51-65)
FUNCTION refresh(玩家):
    清空已加载谱面列表
    清空谱面参数

    // 禁用所有属性文件
    FOR EACH 已启用的脚本:
        IF 脚本名包含 "charts" AND 包含 "_properties":
            禁用脚本

    等待 1 Tick

    // 重新启用所有属性文件
    FOR EACH 已禁用的脚本:
        IF 脚本名包含 "charts" AND 包含 "_properties":
            启用脚本  // 触发 on load 事件，调用 registerChart

    等待 5 Ticks
    显示谱面选择界面

// 2. 谱面注册 (gui.sk:67-77)
FUNCTION registerChart(ID, 歌曲名, 难度, 作曲家, 谱师, 时长, 偏移):
    添加到列表:
        {loadedCharts::id::*} += ID
        {loadedCharts::name::*} += 歌曲名
        {loadedCharts::level::*} += 难度
        {loadedCharts::artist::*} += 作曲家
        {loadedCharts::author::*} += 谱师

    保存参数:
        {length::{ID}} = 时长
        {chartOffset::{ID}} = 偏移

FUNCTION setBPM(ID, BPM):
    {bpm::{ID}} = BPM

// 3. 谱面加载 (gui.sk:18-49)
FUNCTION playChart(谱面ID):
    设置进度 = 0%

    // 禁用其他谱面
    FOR EACH 已启用的脚本:
        IF 脚本名包含 "charts" AND NOT 包含 "_properties":
            禁用脚本

    IF NOT 编辑模式:
        quitGame()  // 清空旧数据

    等待 1 Tick
    设置进度 = 10%

    // 异步加载目标谱面
    异步执行:
        启用脚本 "charts\-{谱面ID}"  // 触发 on load，调用 tap/hold 等函数

    等待完成
    等待 1 Tick

    IF 编辑模式:
        初始化编辑器
    ELSE:
        设置进度 = 25%
        设置当前谱面 = 谱面ID
        执行 /start 命令

// 4. 音符数据注册 (load.sk:7-97)
ON_LOAD:
    创建异步注册段 {asyncNoteRegister}:
        IF NOT 编辑模式:
            保存音符数据到变量:
                {loadedNotes::type::{ID}} = 类型
                {loadedNotes::time::{ID}} = 时间
                {loadedNotes::face::{ID}} = 判定面
                {loadedNotes::x::{ID}} = X坐标
                {loadedNotes::y::{ID}} = Y坐标
                {loadedNotes::glowing::{ID}} = 发光
                {loadedNotes::tag::{ID}} = 标签
                // ... 其他参数
        ELSE:
            转换为编辑器格式并保存

// 各音符注册函数
FUNCTION tap(时间, 判定面, X, Y, 发光, 标签):
    总音符数 += 1
    音符ID += 1
    类型 = "tap"
    异步执行注册段({音符ID}, "tap", {时间}, {判定面}, {X}, {Y}, ..., {发光}, {标签})

FUNCTION hold(时间, 判定面, X, Y, 发光, 标签):
    音符ID += 1  // Hold不计入总音符数（因为不需要点击）
    类型 = "hold"
    异步执行注册段({音符ID}, "hold", ...)

// ... 其他音符类型同理

// 5. 音符排序 (main.sk:225-256)
COMMAND /start:
    设置进度 = 50%

    异步执行:
        // 按时间升序排序所有音符
        排序后的ID列表 = sorted({loadedNotes::time::*}, 升序)

        计数 = 0
        FOR EACH 排序后的ID:
            复制数据到新变量:
                {_sorted::type::{索引}} = {loadedNotes::type::{ID}}
                {_sorted::time::{索引}} = {loadedNotes::time::{ID}}
                // ... 复制所有字段

            计数 += 1
            进度 = 计数 / (音符总数 × 13) × 50 + 50

        删除旧数据 {loadedNotes::*}

        // 重新索引
        FOR EACH 已排序数据:
            {loadedNotes::{字段}::{新索引}} = 值
            计数 += 1
            进度 = 计数 / (音符总数 × 13) × 50 + 50

    等待完成
    进度 = 100%
    开始游戏
```

## 建议的JSON加载实现

### 方案1: 使用 Skript-JSON 插件

#### 安装依赖
```
1. 下载 skript-json.jar
2. 放入 plugins 文件夹
3. 重启服务器
```

#### 加载器实现 (jsonLoader.sk)

```skript
# ============================================
# JSON 谱面加载器
# ============================================

options:
    chartsFolder: plugins/Skript/charts

# 全局变量
# {chartCache::{谱面ID}::metadata::*}
# {chartCache::{谱面ID}::notes::*}

# ============================================
# 谱面列表刷新
# ============================================

function refreshChartList():
    # 清空列表
    delete {loadedCharts::*}
    delete {length::*}
    delete {chartOffset::*}
    delete {bpm::*}

    # 扫描 charts 文件夹
    set {_files::*} to ...  # 需要文件系统插件或脚本

    loop {_files::*}:
        if loop-value ends with ".json":
            set {_id} to loop-value without ".json"
            loadChartMetadata({_id})

# ============================================
# 加载谱面元数据
# ============================================

function loadChartMetadata(chartID: text):
    set {_path} to "{@chartsFolder}/%{_chartID}%.json"
    set {_json} to content of file {_path}

    # 解析 JSON
    set {_data} to json from {_json}

    # 提取元数据
    set {_metadata} to {_data}.get("metadata")

    set {_id} to {_metadata}.get("id")
    set {_name} to {_metadata}.get("name")
    set {_level} to {_metadata}.get("level")
    set {_artist} to {_metadata}.get("artist")
    set {_charter} to {_metadata}.get("charter")
    set {_length} to {_metadata}.get("length")
    set {_offset} to {_metadata}.get("offset")
    set {_bpm} to {_metadata}.get("bpm")

    # 注册谱面
    registerChart({_id}, {_name}, {_level}, {_artist}, {_charter}, {_length}, {_offset})
    setBPM({_id}, {_bpm})

    # 缓存完整数据（可选）
    set {chartCache::%{_id}%} to {_data}

# ============================================
# 加载谱面音符数据
# ============================================

function loadChartNotes(chartID: text):
    # 清空旧数据
    delete {loadedNotes::*}
    set {noteID} to 0
    set {maxNote} to 0

    # 获取缓存的数据
    set {_data} to {chartCache::%{_chartID}%}

    if {_data} is not set:
        set {_path} to "{@chartsFolder}/%{_chartID}%.json"
        set {_json} to content of file {_path}
        set {_data} to json from {_json}

    # 获取音符数组
    set {_notes} to {_data}.get("notes")
    set {_noteArray::*} to json array to list {_notes}

    # 遍历音符
    loop {_noteArray::*}:
        set {_note} to loop-value

        set {_type} to {_note}.get("type")
        set {_time} to {_note}.get("time")

        # 根据类型处理
        if {_type} is "tap":
            parseTapNote({_note})
        else if {_type} is "hold":
            parseHoldNote({_note})
        else if {_type} is "drag":
            parseDragNote({_note})
        else if {_type} is "flick":
            parseFlickNote({_note})
        else if {_type} is "double":
            parseDoubleNote({_note})
        else if {_type} is "execution":
            parseExecutionNote({_note})

# ============================================
# 音符解析函数
# ============================================

function parseTapNote(noteData: object):
    add 1 to {maxNote}
    add 1 to {noteID}

    set {_time} to {_noteData}.get("time")
    set {_face} to {_noteData}.get("face")
```

### 方案2: 使用外部程序预处理

#### Python 转换脚本 `json_to_sk.py`

```python
import json
import os

def convert_json_to_sk(json_path, output_dir):
    """将 JSON 谱面转换为 Skript 格式"""

    with open(json_path, 'r', encoding='utf-8') as f:
        data = json.load(f)

    metadata = data['metadata']
    notes = data['notes']

    chart_id = metadata['id']

    # 生成属性文件
    properties_content = f"""on load:
    registerChart("{chart_id}", "{metadata['name']}", "{metadata['level']}", "{metadata['artist']}", "{metadata['charter']}", {metadata['length']}, {metadata.get('offset', 0)})
    setBPM("{chart_id}", {metadata['bpm']})
"""

    # 生成谱面文件
    notes_content = "on load:\n"

    for note in notes:
        note_type = note['type']

        if note_type in ['tap', 'hold', 'drag']:
            line = f"    {note_type}({note['time']}, \"{note['face']}\", {note['x']}, {note['y']}, {str(note.get('glowing', False)).lower()}, \"{note.get('tag', '')}\")\n"

        elif note_type == 'flick':
            line = f"    flick({note['time']}, \"{note['face']}\", \"{note['turn']}\", {str(note.get('glowing', False)).lower()}, \"{note.get('tag', '')}\")\n"

        elif note_type == 'double':
            line = f"    double({note['time']}, \"{note['face']}\", {note['x1']}, {note['y1']}, {note['x2']}, {note['y2']}, {str(note.get('glowing', False)).lower()}, \"{note.get('tag', '')}\")\n"

        elif note_type == 'execution':
            # 生成 section
            script_name = note['script']
            params = note.get('params', {})

            line = f"""    create section stored in {{{script_name}}}:
        # Custom script: {script_name}
        # Params: {json.dumps(params)}
        send "Execution: {script_name}" to all players
    execution({note['time']}, {{{script_name}}})
"""

        notes_content += line

    # 写入文件
    properties_path = os.path.join(output_dir, f"{chart_id}_properties.sk")
    notes_path = os.path.join(output_dir, f"{chart_id}.sk")

    with open(properties_path, 'w', encoding='utf-8') as f:
        f.write(properties_content)

    with open(notes_path, 'w', encoding='utf-8') as f:
        f.write(notes_content)

    print(f"✓ Converted: {chart_id}")

def batch_convert(json_dir, output_dir):
    """批量转换目录下的所有 JSON 文件"""
    for filename in os.listdir(json_dir):
        if filename.endswith('.json'):
            json_path = os.path.join(json_dir, filename)
            convert_json_to_sk(json_path, output_dir)

if __name__ == '__main__':
    # 使用示例
    batch_convert('charts_json', 'plugins/Skript/scripts/charts')
```

### Skript to JSON 转换器

```python
import re
import json

def parse_sk_to_json(properties_path, notes_path, output_path):
    """将 Skript 谱面转换为 JSON 格式"""

    metadata = {}
    notes = []

    # 解析属性文件
    with open(properties_path, 'r', encoding='utf-8') as f:
        content = f.read()

        # 提取 registerChart
        register_match = re.search(
            r'registerChart\("([^"]+)",\s*"([^"]+)",\s*"([^"]+)",\s*"([^"]+)",\s*"([^"]+)",\s*(\d+),\s*(-?\d+)\)',
            content
        )

        if register_match:
            metadata['id'] = register_match.group(1)
            metadata['name'] = register_match.group(2)
            metadata['level'] = register_match.group(3)
            metadata['artist'] = register_match.group(4)
            metadata['charter'] = register_match.group(5)
            metadata['length'] = int(register_match.group(6))
            metadata['offset'] = int(register_match.group(7))

        # 提取 setBPM
        bpm_match = re.search(r'setBPM\("[^"]+",\s*(\d+(?:\.\d+)?)\)', content)
        if bpm_match:
            metadata['bpm'] = float(bpm_match.group(1))

    # 解析音符文件
    with open(notes_path, 'r', encoding='utf-8') as f:
        content = f.read()

        # 匹配 tap/hold/drag
        for match in re.finditer(
            r'(tap|hold|drag)\(([^,]+),\s*"([^"]+)",\s*([^,]+),\s*([^,]+),\s*(true|false),\s*"([^"]*)"\)',
            content
        ):
            note = {
                'type': match.group(1),
                'time': float(match.group(2)),
                'face': match.group(3),
                'x': float(match.group(4)),
                'y': float(match.group(5)),
                'glowing': match.group(6) == 'true',
                'tag': match.group(7)
            }
            notes.append(note)

        # 匹配 flick
        for match in re.finditer(
            r'flick\(([^,]+),\s*"([^"]+)",\s*"(left|right)",\s*(true|false),\s*"([^"]*)"\)',
            content
        ):
            note = {
                'type': 'flick',
                'time': float(match.group(1)),
                'face': match.group(2),
                'turn': match.group(3),
                'glowing': match.group(4) == 'true',
                'tag': match.group(5)
            }
            notes.append(note)

        # 匹配 double
        for match in re.finditer(
            r'double\(([^,]+),\s*"([^"]+)",\s*([^,]+),\s*([^,]+),\s*([^,]+),\s*([^,]+),\s*(true|false),\s*"([^"]*)"\)',
            content
        ):
            note = {
                'type': 'double',
                'time': float(match.group(1)),
                'face': match.group(2),
                'x1': float(match.group(3)),
                'y1': float(match.group(4)),
                'x2': float(match.group(5)),
                'y2': float(match.group(6)),
                'glowing': match.group(7) == 'true',
                'tag': match.group(8)
            }
            notes.append(note)

    # 按时间排序
    notes.sort(key=lambda n: n['time'])

    # 生成 JSON
    output_data = {
        'metadata': metadata,
        'notes': notes
    }

    with open(output_path, 'w', encoding='utf-8') as f:
        json.dump(output_data, f, indent=2, ensure_ascii=False)

    print(f"✓ Converted to JSON: {output_path}")

def batch_convert_sk_to_json(charts_dir, output_dir):
    """批量转换 Skript 谱面为 JSON"""
    import os

    chart_files = {}

    # 收集谱面文件对
    for filename in os.listdir(charts_dir):
        if filename.endswith('_properties.sk'):
            chart_id = filename.replace('_properties.sk', '')
            chart_files[chart_id] = {'properties': os.path.join(charts_dir, filename)}
        elif filename.endswith('.sk'):
            chart_id = filename.replace('.sk', '')
            if chart_id not in chart_files:
                chart_files[chart_id] = {}
            chart_files[chart_id]['notes'] = os.path.join(charts_dir, filename)

    # 转换
    for chart_id, files in chart_files.items():
        if 'properties' in files and 'notes' in files:
            output_path = os.path.join(output_dir, f"{chart_id}.json")
            parse_sk_to_json(files['properties'], files['notes'], output_path)

if __name__ == '__main__':
    # 使用示例
    batch_convert_sk_to_json(
        'plugins/Skript/scripts/charts',
        'charts_json'
    )
```

## 总结

### 当前 Skript 格式
✅ 优点：
- 与游戏系统深度集成
- 可以执行任意代码（Execution 音符）
- 加载时自动执行

❌ 缺点：
- 需要 Skript 知识
- 格式验证困难
- 不利于版本控制
- 难以开发外部工具

### 建议的 JSON 格式
✅ 优点：
- 通用性强
- 易于验证
- 工具支持好
- 可读性高
- 扩展性强

❌ 缺点：
- 需要额外的加载器
- Execution 音符需要预定义
- 可能需要额外依赖

### 推荐方案
**使用 JSON 格式 + Skript 加载器**

原因：
1. JSON 便于谱面制作和管理
2. Skript 加载器易于实现
3. 保持游戏系统不变
4. 便于未来扩展（如外部编辑器）

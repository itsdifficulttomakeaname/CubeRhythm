## 渲染系统概述

### 核心组件

**位置**: `main.sk` + `load.sk`

**主要实体类型**:
1. **Block Display** - 音符可视化方块
2. **Interaction** - 音符判定碰撞体
3. **Text Display** - 箭头指示器（Flick音符）

### 渲染循环

**触发频率**: 每Tick（20次/秒）
**位置**: `main.sk` 第293-418行

```skript
every ticks:
    {timer} is set
    # 1. 检测是否需要生成新音符
    # 2. 计算音符距离
    # 3. 调用对应的draw函数
    # 4. 更新现有音符位置
```

**JSON配置描述**:
```json
{
  "renderLoop": {
    "tickRate": 20,
    "maxEntities": 100,
    "preloadDistance": 50,
    "despawnDistance": 4,
    "updateFrequency": "every_tick"
  }
}
```

## 音符生成机制

### 距离计算公式

**位置**: `main.sk` 第308行

```skript
set {_tempDistance} to {speed} * 20 * ({loadedNotes::time::%loop-index%} - ({timer} / 20)) + 4
```

**公式解析**:
- `{speed}`: 音符速度倍率（默认1.0）
- `20`: Tick转秒的转换率
- `{loadedNotes::time}`: 音符时间（秒）
- `{timer} / 20`: 当前播放时间（秒）
- `+ 4`: 判定面偏移量

**JSON配置**:
```json
{
  "distanceCalculation": {
    "formula": "speed * tickRate * (noteTime - currentTime) + offset",
    "parameters": {
      "speed": {
        "default": 1.0,
        "min": 0.5,
        "max": 2.0,
        "description": "音符速度倍率"
      },
      "tickRate": {
        "value": 20,
        "description": "每秒Tick数"
      },
      "offset": {
        "value": 4,
        "description": "判定面距离偏移"
      }
    },
    "threshold": {
      "summon": 50,
      "description": "小于此距离时生成音符"
    }
  }
}
```

### 生成条件

**位置**: `main.sk` 第305-319行

```skript
if {_count} < 100:  # 实体数量限制
    loop {loadedNotes::type::*}:
        set {_tempDistance} to ...
        add 1 to {_preSummon} if {_tempDistance} < 50
```

**JSON条件配置**:
```json
{
  "spawnConditions": {
    "maxEntities": {
      "value": 100,
      "type": "interaction",
      "description": "最大交互实体数量"
    },
    "spawnDistance": {
      "value": 50,
      "unit": "blocks",
      "description": "生成距离阈值"
    },
    "despawnDistance": {
      "value": 4,
      "unit": "blocks",
      "description": "判定面位置（触及后删除）"
    }
  }
}
```

**函数位置**: `load.sk` 第98-143行

**渲染步骤**:
```skript
function drawTap(face, x, y, z, glowing, tag):
    # 1. 计算位置
    # 2. 根据判定面旋转坐标
    # 3. 生成Block Display
    # 4. 生成Interaction碰撞体
    # 5. 应用平移动画
```

**JSON渲染配置**:
```json
{
  "tap": {
    "visualEntity": {
      "type": "block_display",
      "block": "light_blue_concrete",
      "scale": {"x": 1, "y": 1, "z": 0.5},
      "interpolation": {
        "start": 0,
        "duration": 100
      },
      "brightness": {
        "block": 15,
        "sky": 15
      },
      "animation": {
        "type": "translation",
        "target": {"x": 0, "y": 0, "z": -100},

      }
    },
    "hitboxEntity": {
      "type": "interaction",
      "size": {"width": 2, "height": 2},
      "position": {"x": "+0.5", "y": "-0.5", "z": "+1"},
      "metadata": {
        "notetype": "tap",
        "face": "from_parameter",
        "linkuuid": "block_display_uuid",
        "tag": "from_parameter"
      }
    },
    "glowEffect": {
      "enabled": "from_parameter",
      "color": {"r": 0, "g": 50, "b": 255, "a": 255}
    },
    "faceRotations": {
      "w": {"yaw": 0, "pitch": 0, "translation": {"z": -100}},
      "a": {"yaw": 90, "pitch": 0, "translation": {"z": 100}},
      "s": {"yaw": 180, "pitch": 0, "translation": {"z": -100}},
      "d": {"yaw": 270, "pitch": 0, "translation": {"z": 100}}
    }
  }
}
```

**坐标转换规则**:

| Face | Block Display 位置 | Interaction 位置 | 旋转角度 |
|------|-------------------|------------------|---------|
| w    | `(x, y, z)`       | `(x+0.5, y-0.5, z+1)` | 0° |
| a    | `(z, y, -x)`      | `(z+1, y-0.5, -x)` | 90° |
| s    | `(-x+1, y, -z+1)` | `(-x, y-0.5, -z)` | 180° |
| d    | `(-z+1, y, x+1)`  | `(-z, y-0.5, x+0.5)` | 270° |
